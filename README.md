# Simple Pendulum Simulation

This project simulates the motion of a simple pendulum using Python.

## Features
- Solves pendulum motion using physics equations
- Plots angle vs time graph
- Visualizes pendulum movement through animation

## Concepts Used
- Simple Harmonic Motion (SHM)
- Angular frequency
- Numerical simulation

## Tools Used
- NumPy
- Matplotlib

## Output
The simulation shows oscillatory motion and a real-time animated pendulum.
